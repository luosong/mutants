Mutants

Mutants 是一个用来开发H5应用的框架,基于React的页面显示。也可以用来作为原型开发工具,开始创建一个应用的原型。

Mutants 包含了chanjet-mutants,chanjet-navigator,chanjet-scroller,chanjet-ui,chanjet-virtual-list,chanjet-sdk等一系列的模块,涵盖了全方面的内容。

如果你想在工作圈中开发IOS或者Android混合应用,Mutants将会是你的首选。

简单,快速

只需三步即可生成可以正常运行的项目结构,方法简单,搭建快速。

$ npm install chanjet-gzq-h5 -g 
$ chanjet-gzq-h5 myapp
$ cd myapp & npm install

使用生成项目的工具,非常简单。你只需要确保电脑上安装了node及npm(建议使用node版本4.x)。在生成的项目中,你可以根据需求使用对应的模块即可,自由组合使用!并且生成的项目中,已使用webpack将文件的打包,压缩的配置项都准备就绪。只需要启动服务就可以啦!

UI 组件

chanjet-ui 中有很多可以直接使用的UI组件,比如List,Button,Dialog,TextField,Menu等等。Mutants 是针对于移动应用的框架,因此除了有丰富的UI组件外,还考虑到了如何适配不同手机屏幕分辨率,提供了三种适配方案(px,rpx,dpx),让UI组件的更好的展示,达到更优的效果。

Plugins

多样的插件,也是Mutants的特点之一。提供了包括定位、拍照、读取相册、电话、短信、语音识别、预览、分享等的插件功能。加强了与工作圈的能力的交互,使用方式亦简单化。
